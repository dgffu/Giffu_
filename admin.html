Configuração da API do Google / YouTube

Como funciona a autenticação?
Para enviar vídeos diretamente para o canal Giffú via API, você precisa fornecer o seu OAuth 2.0 Client ID criado no Google Cloud Console com a conta dilangoficial@gmail.com.

Instruções para criar o Client ID no Google Cloud Console:

  1. Acesse o Google Cloud Console logado na sua conta dilangoficial@gmail.com.
  2. Crie um projeto ou selecione um existente (ex: Giffu Portfolio).
  3. No menu lateral, vá em APIs e Serviços > Biblioteca e ative a YouTube Data API v3.
  4. Vá em APIs e Serviços > Credenciais > Clique em Criar Credenciais > ID do cliente OAuth.
  5. Em Tipo de Aplicação, escolha Aplicação Web.
  6. Em Origens JavaScript autorizadas, adicione:
    https://giffu.com.br e http://localhost:8080 (ou seu domínio local).
  7. Copie o Client ID gerado e cole no campo acima!
